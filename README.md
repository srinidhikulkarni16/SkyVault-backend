**SkyVault – Backend**

 📌 Description

SkyVault backend is a scalable REST API designed to handle secure file storage, access control, and high-performance file operations in a cloud environment.


 🚀 Features

* JWT-based authentication (access & refresh tokens)
* File & folder CRUD operations
* Secure multipart file uploads (presigned URLs)
* Role-based access control (Owner, Editor, Viewer)
* Public file sharing with expiry and password protection
* Search and filtering capabilities
* Trash and restore system


 🧱 Tech Stack

* Node.js
* Express.js
* PostgreSQL (Supabase)
* Supabase Storage
